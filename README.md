# Pentagon.Extensions.Logging

[![NuGet][nuget-badge]][nuget]  

Pentagon.Extensions.Logging is a library for .NET applications. Provides extension logic for Microsoft.Extensions.Logging and Serilog.

[nuget]: https://www.nuget.org/packages/Pentagon.Extensions.Logging/
[nuget-badge]: https://img.shields.io/nuget/v/Pentagon.Extensions.Logging.svg?style=flat

## Install via NuGet

```
PM> Install-Package Pentagon.Extensions.Logging
```
